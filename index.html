Conway's Game of Life trifft Ravels Boléro

Das unten abgebildete Orchester soll das Stück "Boléro" von Maurice Ravel spielen. Dieses besteht aus einer sich immer wieder wiederholenden Abfolge zweier Themen, lediglich die Instrumentation ändert sich dabei.

Auf diesem Ablauf fußt auch der Gedanke des hier zu sehenden Programms: Die Abfolge der Themen bleibt immer gleich, auf ein zweitaktiges Intro folgen das in zwei Hälften aufgeteilte A-Thema, erneut zwei Zwischentakte und schließlich das zweigeteilte B-Thema, woraufhin der Ablauf wieder von vorne beginnt. Die in den einzelnen Abschnitten aktiven Instrumente werden dabei durch eine angepasste Version des Algorithmus "Conway's Game of Life" bestimmt. Beginnend mit einem bestimmten Start-Ensemble werden vor jedem Abschnittswechsel die Nachbarn jedes Instruments überprüft, wobei "Über-Eck-Verbindungen" auch als Nachbarschaft zählen. Hat ein aktives Instrument 1-2 ebenfalls aktive Nachbarn, so bleibt es weiterhin aktiv, andernfalls hört es auf zu spielen. Hat ein inaktives Instrument mindestens 2 aktive Nachbarn, so wird es aktiv.

Der Ablauf des Stücks kann nun vom Benutzer gesteuert werden. Per Mausklick auf die jeweiligen Felder können Instrumente dem Start-Ensemble hinzugefügt werden. Mit dem Reset-Button kann diese Auswahl zurückgesetzt werden. Ist die Auswahl abgeschlossen, so kann mittels des Start-Buttons der Algorithmus gestartet werden. Dieser Button kann nach dem Start dazu genutzt werden, um den Ablauf, beispielsweise im Falle einer sehr langen oder unendlichen Folge, wieder zu stoppen. In diesem Fall wird das Orchester den jeweils aktuellen Abschnitt noch zuende spielen und schließlich aufhören.
